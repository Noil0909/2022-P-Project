# 2022 P-Project (FE)
블록체인을 활용한 블랙박스 서비스(FE)
## 팀원
- BE: 김석준, 유관우
- FE: 김정윤, 이태희
## 프로젝트 개요
### 개발의 필요성
- 현재 대한민국 대부분의 차량은 블랙박스 장착함
- 기존 차량용 블랙박스는 쉽게 고장남
- 일부 사고는 블랙박스만으로는 쉽게 사고의 책임을 가리기 힘든 한계
### 개발 목표
- 기존 블랙박스 기능은 유지
- 블랙박스가 촬영한 영상을 블록체인 네트워크에 업로드
- 차량 내 센서데이터도 같이 블록체인 네트워크에 업로드
## UI 설계
- 로그인 및 회원가입 기능
- 메인 화면
- 일반 조회 기능
- 사고 조회 기능
- 개인 정보 조회 및 수정 기능
